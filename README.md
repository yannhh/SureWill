SureWill Cryptographic Digital Vault - Setup and Execution Guide

Prerequisites:
Before running the SureWill Web App, ensure your system has these installed:

1. Node.js v16 (or higher)
2. npm (node package manager) usually comes with Node.js

Step 1: Install Backend Dependencies:

1. cd path to the server.
2. "npm install" in terminal.

Step 2: Install Frontend Dependencies:

1. cd path to surewill-frontend
2. "npm install" in terminal.

Step 3: Starting Web App

1. npm run dev in terminal.
2. npm start (for frontend) in terminal.
